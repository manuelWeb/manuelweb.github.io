Démo Github Copilot

Qu'est-ce que Github Copilot ?

GitHub Copilot est une extension pour Visual Studio Code qui vous aide à écrire du code plus rapidement.

Comment fonctionne Github Copilot ?

GitHub Copilot est basé sur Codex, un modèle de langage généré par OpenAI. Il fonctionne en analysant le code que vous écrivez et en générant des suggestions de code en temps réel.

Installation et coût de GitHub Copilot

GitHub Copilot est une extension gratuite pour Visual Studio Code. Vous pouvez l'installer directement depuis le Marketplace de Visual Studio Code.

Pour l'installer :

  1. Ouvrez Visual Studio Code.
  2. Cliquez sur l'icône des extensions dans la barre latérale gauche (ou utilisez le raccourci `Cmd+Shift+X` sur Mac).
  3. Dans la barre de recherche des extensions, tapez "GitHub Copilot".
  4. Cliquez sur le bouton "Installer".

Une fois installé, GitHub Copilot est activé par défaut et commencera à fournir des suggestions de code automatiquement.
